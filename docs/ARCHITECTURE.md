# Projekt-Architektur

## Übersicht

Dieses Website-Projekt folgt einer modularen, komponentenbasierten Architektur.

## Schichten

### 1. Präsentationsschicht (Frontend)
- HTML Templates
- CSS Styles
- JavaScript Komponenten

### 2. Assets-Schicht
- Bilder
- Fonts
- Icons

### 3. Build-System
- Vite als Build Tool
- ESLint für Code Quality
- Prettier für Code Formatting

## Best Practices

- Komponenten sollten wiederverwendbar sein
- CSS sollte modular organisiert sein
- JavaScript sollte in Module aufgeteilt sein
- Alle Assets sollten optimiert sein
