# Website Project

Ein modernes Website-Projekt mit einer klaren Projektstruktur.

## 📁 Projektstruktur

```
├── src/                    # Quellcode
│   ├── index.html         # Haupteinstiegspunkt
│   ├── styles/            # CSS/SCSS Dateien
│   ├── scripts/           # JavaScript/TypeScript Dateien
│   ├── components/        # Wiederverwendbare Komponenten
│   └── assets/            # Bilder, Fonts, etc.
├── public/                # Statische Dateien
├── tests/                 # Test-Dateien
├── docs/                  # Dokumentation
├── config/                # Konfigurationsdateien
├── scripts/               # Build- und Utility-Skripte
├── package.json          # Projekt-Abhängigkeiten
├── README.md            # Projekt-Dokumentation
└── .gitignore           # Git-Ignorierungsdatei
```

## 🚀 Getting Started

### Installation

```bash
npm install
```

### Entwicklung

```bash
npm run dev
```

Die Website läuft unter `http://localhost:5173`

### Build

```bash
npm run build
```

### Vorschau

```bash
npm run preview
```

## 🔧 Verfügbare Commands

- `npm run dev` - Startet den Entwicklungsserver
- `npm run build` - Erstellt die optimierte Build-Version
- `npm run preview` - Zeigt eine Vorschau der Build-Version
- `npm run lint` - Führt Linting durch
- `npm run format` - Formatiert den Code
- `npm run test` - Führt Tests aus
