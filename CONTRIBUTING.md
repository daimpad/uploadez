# Contributing

Vielen Dank für Ihr Interesse an diesem Projekt!

## Richtlinien

1. **Code-Style**: Verwenden Sie `npm run format` für Code-Formatierung
2. **Linting**: Führen Sie `npm run lint` aus, bevor Sie committen
3. **Tests**: Schreiben Sie Tests für neue Funktionen
4. **Dokumentation**: Aktualisieren Sie die Dokumentation bei Änderungen

## Workflow

1. Fork das Repository
2. Erstellen Sie einen Feature-Branch
3. Committen Sie Ihre Änderungen
4. Push zu Ihrem Fork
5. Erstellen Sie einen Pull Request

## Code of Conduct

Bitte seien Sie respektvoll und konstruktiv in allen Interaktionen.
